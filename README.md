1️⃣ What is the difference between var, let, and const?
answe: 
    var is a scoped function which sometimes makes bugs because it ignores block{}.

    let is block scoped which is stays inside {}. Use it for values that can change.

    const is also block scoped. It don't allow to reassign. use const for values that shouldn't change.

2️⃣ What is the spread operator (...)?
answer:
    it's like spreading things out.it use with arrays, objects or function arguments.

3️⃣ What is the difference between map(), filter(), and forEach()?
answer:
    map() makes a new array by transforming each item.

    filter() makes a new array but only with items that pass a condition.

    forEach() just goes through each item, doesn't return anything.

4️⃣ What is an arrow function?
